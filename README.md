# mp3-recorder
日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A simple MP3 recorder web application.

## Demo
[https://code4fukui.github.io/mp3-recorder/](https://code4fukui.github.io/mp3-recorder/)

## Features
- Record audio in the browser and encode it as MP3 files
- Adjust the MP3 encoding quality (VBR 0-9)
- Play back the recorded audio
- Download the recorded MP3 file

## Usage
1. Open the web application in a compatible browser.
2. Select the desired MP3 encoding quality from the dropdown.
3. Click the "Record" button to start recording audio.
4. Click the "Stop" button to stop recording.
5. Click the "Play" button to listen to the recorded audio.
6. Click the "Download" button to save the recorded MP3 file.
