# mp3-recorder
[README.ja.md](README.ja.md)にある日本語のREADMEをご覧ください。

シンプルなMP3レコーダーのWebアプリケーションです。

## デモ
[https://code4fukui.github.io/mp3-recorder/](https://code4fukui.github.io/mp3-recorder/)

## 機能
- ブラウザで音声を録音し、MP3ファイルとしてエンコードする
- MP3エンコーディング品質を調整する(VBR 0-9)
- 録音した音声を再生する
- 録音したMP3ファイルをダウンロードする

## 使用法
1. 互換性のあるブラウザでウェブアプリケーションを開きます。
2. ドロップダウンメニューから希望のMP3エンコーディング品質を選択します。
3. 「録音」ボタンをクリックしてオーディオの録音を始めます。
4. 「停止」ボタンをクリックして録音を停止します。
5. 「再生」ボタンをクリックして録音したオーディオを聞きます。
6. 「ダウンロード」ボタンをクリックして録音したMP3ファイルを保存します。
