# MP3レコーダー

MP3ファイルの録音、再生、ダウンロードができるWebアプリケーションです。ブラウザ上で手軽に音声を記録することができます。

## デモ
https://code4fukui.github.io/mp3-recorder/

## 機能
- マイクからの音声入力
- MP3形式での録音
- 録音したMP3の再生
- 録音したMP3のダウンロード

## 使い方
1. ブラウザ画面を開く
2. マイクへのアクセスを許可する
3. 「録音」ボタンをクリックして録音を開始する
4. 「停止」ボタンをクリックして録音を終了する
5. 「再生」ボタンをクリックして録音した音声を試聴する
6. 「ダウンロード」ボタンをクリックして録音したMP3ファイルをパソコンに保存する
