# MP3レコーダー / mp3rec

MP3ファイルの録音・再生・ダウンロードができるWebアプリケーションです。ブラウザベースで手軽に音声を記録できます。

## デモ
https://code4fukui.github.io/mp3-recorder/

## 機能
- マイクからの音声入力
- MP3形式での録音
- 録音したMP3の再生
- 録音したMP3のダウンロード

## 使い方
ブラウザ上で動作します。ページを開くと、マイクへのアクセスを求められるので許可してください。

「録音」ボタンをクリックすると録音が開始され、「停止」ボタンで録音を終了します。
録音したMP3ファイルは「再生」ボタンで試聴でき、「ダウンロード」ボタンでパソコンに保存できます。
